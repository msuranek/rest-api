# PHP 8 REST API Example

## Požadavky

- PHP 8.0+
- Composer

## Instalace

```bash
composer install
```
## Spuštění serveru

```bash
php -S localhost:8000 -t example
```

## API Endpoint

```bash
GET /api/user — vrací JSON uživatele
```

## Testy

Spustíš příkazem:
```bash
vendor/bin/phpunit
```
